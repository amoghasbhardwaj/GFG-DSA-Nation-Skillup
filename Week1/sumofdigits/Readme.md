# Sum of Digits

## Problem Statement

Given an integer `n`, return the sum of its digits.

---

## Input

- A single integer `n` (1 ≤ n ≤ 10⁶)

## Output

- A single integer: the sum of the digits of `n`.

---

## Examples

### Example 1  
**Input:**  
`687`  
**Output:**  
`21`  
**Explanation:** 6 + 8 + 7 = 21

### Example 2  
**Input:**  
`1234`  
**Output:**  
`10`  
**Explanation:** 1 + 2 + 3 + 4 = 10
